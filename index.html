MY WORK

PODFAST

A Progressive Web Application for mobile, which helps people on the go find and choose podcast episodes quickly.

Built with Ruby on Rails, JavaScript, HTML, CSS/SCSS, Bootstrap, AJAX, PostgreSQL, ActiveRecord, SQL, Spotify/ListenNotes APIs, PG_search, Acts as votable, Acts as Followable, Social-Share, Figma (prototype and wireframe)

Browse

The app home page, where users can find popular podcasts and browse new releases.

Users can also click a button to pick a random episode or see episodes their friends have listened to

Search

Introducing more flexibility into the users search experience, with the option to select multiple categories, moods and set a duration in their preferences

Results

Episodes are sorted by relevace and can be filtered to show most liked and listened to.

Users can also see how many of their friends have liked each episode

Episodes

Users can listen, like, comment, share and add episodes to 'pod'lists.

Podlists

'Pod'lists allow users to save multiple episodes from a search for later

MY WORK

SWIPE CHEF

Recipe speed dating for uninspired foodies. Think Tinder but for recipes!

Our recipes are scraped from a number of go-to food websites, including BBCgoodfood and are presented using JS carousels

Landing

A simple, well designed landing page that encourages users to click or swipe through to find their recipe match

Swipe right

A dating app style approach to chosing recipes, based on user preferences for cooking time and food categories.

Let's cook

A one page summary of the recipe, for ease of navigation while cooking. Users can navigate through each step using the arrow keypad or by clicking on the arrows.

MY WORK

NOMAD

A Ruby on Rails Web Application based on AirBNB, for adventurers looking for unique accommodation in highly remote locations

Built with Ruby on Rails, JavaScript, HTML, CSS/SCSS, Bootstrap, PostgreSQL, ActiveRecord, SQL, Devise, Geocoder gem, Mapbox API, Algolia places API.

Search

The home page of the app, where users can search for accomodation by location

List

Accomodation is listed adjacent to a map, which has markers and info cards to show facilities and distance from the search address

Choose

Users can see more details of the selected accomodation, including reviews and request to book

Dashboard

Users can see their upcoming bookings and review previous stays

3/3
